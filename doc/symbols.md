Symbols
-------
---

Mara defines the following symbols.  Notice that all but a few symbols are mapped to only one meaning regardless of context.

```
*-------*-----------*---------------------*
| `+`   | `plus`    | Addition            |
| `-`   | `minus`   | Subtraction         |
| `*`   | `times`   | Multiplication      |
| `/`   | `divide`  | Division            |
| `^`   | `power`   | Power               |
*-------*-----------*---------------------*
| `::`  | `Bind`    | Bind                |
| `|`   | `Variant` | Variant Type        |
| `&`   | `Compose` | Type Composition    |
*-------*-----------*---------------------*
| `.`   | `call`    | Funcion Application |
*-------*-----------*---------------------*
| `++`  | `concat`  | Concatenate/Append  |
| `--`  | `remove`  | Remove/Diference    |
*-------*-----------*---------------------*
```

## Declarations

```
*---------*-------------*
| `def`   | Function    |
| `val`   | Value       |
| `var`   | Variable    |
| `let`   | Alias       |
*---------*-------------*
| `trait` | Trait       |
| `object` | Object     |
| `proto` | Protocol    |
*---------*-------------*
```

## Expression Keywords

```
*---------------*-----------------------*
| `for` `in`          | Loop Exprs      |
| `match` `as`        | Match Exprs     |
| `while`             | Simple Loop     |
| `if` `elif` `else`  | Conditionals    |
| `do`                | Eval Exprs      |
*---------------------*-----------------*
```

## Operators

```
*-------*---------------*
| `||`  |  boolean or   |
| `&&`  |  boolean and  |
| `^^`  |  boolean xor  |
| `!`   |  boolean not  |
*-------*---------------*
| `~|`  | binary or     |
| `~!`  | binary not    |
| `~&`  | binary and    |
| `~>>` | shift right   |
| `~<<` | shift left    |
*-------*---------------*
```
